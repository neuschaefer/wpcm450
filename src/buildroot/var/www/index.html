lolbmc (PREVIEW)

This a baseboard management controller (BMC) with custom firmware based on familiar Unix tools.

Unlike typical BMC firmware, it doesn't implement IPMI or a fancy web interface, or rely on Java/Flash applets. Instead, the protocol of choice is plain old SSH.

Power management

ssh bmc@host power on

Serial over LAN

SOL can be entered by typing sol on the shell, or through ssh sol@host

Graphical console

There is a VNC server listening on localhost:5900, which can be accessed through an SSH tunnel.

The source code for this firmware is available on GitHub (neuschaefer/wpcm450).
